
# PET SHOP

This is a pet shop e-commerce Backend (API)


## Installation

### Docker

```shell
make all
```

## Test and Analyse

```shell
make test
```
```shell
make insight
```
```shell
make analyse
```

### Interact with docker image
App App Image
```shell
make app
```
DB Image
```shell
make db
```
Seed data
```shell
make seed
```
